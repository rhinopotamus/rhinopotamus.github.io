Compute \(\dfrac{d}{dx}\left(f-y'\dfrac{\partial f}{\partial y'}\right)\text{.}\) Use the chain rule to write out \(\dfrac{df}{dx}\text{.}\) Contrary to how the book is going to do it in the next paragraph, don't write out \(\dfrac{d}{dx} \dfrac{\partial f}{\partial y'}\text{!}\) Simplify a little; you should in particular be able to factor out \(y'\text{.}\)
